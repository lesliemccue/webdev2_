The purpose of my website is to showcase how the use of social media apps can aid lectures in teaching. I plan on creating an easy to use website that is very clear and precise, I want it to be easily understood. If I achieve this lectures could use my website to research different apps and see how they could be used for learning purposes. A how to use page will also hopefully set up explain how to download, sign up and use the app which will be very helpful. I will also aim to identify limitations of certain apps so the user can make an informed decision on what app best suits them. I will include videos and also screenshots showing processes, and hopefully some user reviews.
